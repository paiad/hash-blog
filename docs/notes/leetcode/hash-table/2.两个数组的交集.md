给定两个数组 nums1 和 nums2 ，返回它们的交集 。输出结果中的每个元素一定是唯一的。我们可以不考虑输出结果的顺序 。



>示例 1：
输入：nums1 = [1,2,2,1], nums2 = [2,2]
输出：[2]

>示例 2：
输入：nums1 = [4,9,5], nums2 = [9,4,9,8,4]
输出：[9,4]
>
>解释：[4,9] 也是可通过的

```c++
class Solution {
public:
    vector<int> intersection(vector<int>& nums1, vector<int>& nums2) {
        vector<int> res;
        vector<int> buckets(1001,0);

        for (int num1:nums1){
            buckets[num1] = 1;
        }

        for (int num2:nums2){
            if(buckets[num2] == 1){
                res.push_back(num2);
                buckets[num2] = 0;
            }
        }

        return res;
    }
};
```
